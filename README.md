# Maze · Simulador Growth Engine Ops

Simulador predictivo a 6 meses del Growth Engine Ops de Maze. Permite jugar con
CAC, retención, tiers, mix de clientes e infraestructura IA para proyectar
ingresos, costos y utilidad mes a mes.

## Qué es

Una aplicación **100% estática** (un solo `index.html` con HTML + CSS + JavaScript).
No tiene backend ni base de datos: todo el cálculo corre en el navegador.

## Cómo usarlo

Abrí `index.html` en cualquier navegador. No requiere instalación ni servidor.

## Estructura

- `index.html` — la app completa (interfaz, modelo de cálculo y gráficos).

## Stack

- HTML / CSS / JavaScript puro (sin frameworks ni dependencias).
- Tipografías: Fraunces + Hanken Grotesk (Google Fonts).
